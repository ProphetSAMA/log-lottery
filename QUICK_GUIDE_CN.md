## 输入示例

### 示例 1：指定3个人保底
```
U001,U002,U003
```

### 示例 2：指定5个人保底（带空格也可以）
```
U001, U002, U003, U010, U020
```

### 示例 3：只指定1个人
```
U001
```

## 效果

输入后，下次抽奖时：
1. ✅ **首先**：U001、U002、U003 会优先被抽中（手动保底）
2. ✅ **其次**：连续未中奖5次的人会被抽中（自动保底）
3. ✅ **最后**：从剩余人员中随机抽取

## 快速操作步骤

1. 打开应用
2. 点击顶部导航的 **[全局配置]**
3. 在页面中找到 **[功能设置]** 卡片
4. 确保 **[保底匹配]** 复选框已勾选 ✓
5. 在 **[保底人员编号]** 输入框中输入人员编号
6. 返回首页进行抽奖
7. ✅ 完成！

## 常见问题

**Q: 找不到"功能设置"？**
A: 它在全局配置页面的下方，是一个独立的卡片区域，标题是"功能设置 (Ability Setting)"

**Q: 输入后没有效果？**
A: 请检查：
- 保底匹配是否已启用（复选框要勾选）
- 输入的编号是否正确（在人员列表中存在）
- 编号格式是否正确（用英文逗号分隔）

**Q: 怎么知道人员的编号（uid）？**
A: 在"人员配置"页面的"编号"列可以看到每个人的uid

**Q: 抽奖后需要清空编号吗？**
A: 建议每次抽奖后清空，避免重复使用
